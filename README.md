# 🎵 SCloudMusicBot — Умный музыкальный бот

![Status](https://img.shields.org/badge/Status-Active-success)
![Node](https://img.shields.org/badge/Node.js-v20+-green)
![DB](https://img.shields.org/badge/DB-Supabase-blue)

Продвинутый Telegram-бот для поиска и скачивания музыки с SoundCloud (и в будущем других платформ). Бот отличается высокой стабильностью, умным поиском с исправлением опечаток и собственной системой кэширования.

---

## 🔥 Ключевые особенности

### 🎧 Скачивание и Обработка
*   **SoundCloud Downloader:** Скачивание треков в максимальном доступном качестве (MP3/Opus).
*   **Resilient System:** Гибридная система загрузки (`scdl` + `yt-dlp`). Если один метод не срабатывает, бот автоматически использует резервный.
*   **Плейлисты:** Поддержка скачивания целых альбомов и плейлистов с пагинацией.

### 🧠 Умный Поиск (Smart Search)
*   **Гибридный поиск:** Мгновенный поиск по локальному кэшу + Живой поиск в SoundCloud.
*   **Fuzzy Search:** Бот понимает опечатки и транслит.
    *   *Вводите:* `адисса`, `федук`, `небо славяян`
    *   *Находит:* `Алиса`, `Feduk`, `Небо Славян`
*   **Trigram Similarity:** Использование алгоритмов PostgreSQL (`pg_trgm`) для нечеткого сравнения строк.

### ⚡ Производительность
*   **Кэширование:** Скачанные треки сохраняются в канале-хранилище и базе данных. Повторная выдача занимает 0.1 сек.
*   **Redis:** Используется для сессий и быстрых операций.
*   **Очередь (Task Queue):** Умное распределение нагрузки, приоритеты для Premium-пользователей.

### 📊 Админка и Дашборд
*   Веб-интерфейс для просмотра статистики.
*   Графики активности, топы запросов, управление пользователями.
*   Система рассылок (Broadcast) с поддержкой медиа.

## 🚀 Установка и запуск (Dev)

1.  **Клонирование:**
    ```bash
    git clone https://github.com/YourUsername/SCloudMusicBot.git
    ```

2.  **Установка зависимостей:**
    ```bash
    npm install
    ```

3.  **Настройка переменных окружения (`.env`):**
    ```env
    BOT_TOKEN=ваш_токен
    DATABASE_URL=ссылка_на_supabase
    REDIS_URL=ссылка_на_redis
    ADMIN_ID=ваш_id
    ```

4.  **Запуск:**
    ```bash
    npm start
    ```
